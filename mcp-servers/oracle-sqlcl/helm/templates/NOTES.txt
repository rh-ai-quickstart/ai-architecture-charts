⚠️  DEPRECATED: Oracle SQLcl MCP Server Standalone Chart

This standalone oracle-sqlcl chart has been deprecated and is no longer recommended.

🚀 MIGRATION REQUIRED:

Oracle SQLcl MCP is now integrated into the unified mcp-servers chart.

Please use the parent mcp-servers chart instead:

1. Uninstall this chart:
   helm uninstall {{ .Release.Name }} -n {{ .Release.Namespace }}

2. Install the unified mcp-servers chart:
   helm install mcp-servers ../helm -n {{ .Release.Namespace }}

3. Enable Oracle SQLcl in the unified chart:
   --set mcp-servers.oracle-sqlcl.mcpserver.enabled=true

Benefits of the unified approach:
✅ Centralized Toolhive operator management
✅ MCPServer CRDs instead of Deployments
✅ Secure credential management via Kubernetes secrets
✅ Better resource utilization
✅ Simplified operations

For more information, see the mcp-servers README.md
